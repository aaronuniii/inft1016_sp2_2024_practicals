UNISA WEB PAGE ASSIGNMENT 2
In this assignment, the requirements was to create a web page that included the following steps that was provided.
The website is created on VIsual Code using HTML, JavaScript and CSS.
The main features used in this website were --
1. Toggle Theme: Where users can click a button which includes a dark mode and light mode change in theme.
2. Picture Change: There are two buttons in the first page and it includes the ability to change between two correlating features while also adding comments.
3. Courses: Functioning buttons that will lead you to the course page and when clicking on the links it will send you to the course website.
4. To Do List: A functioning list with specific features.
5. Dynamic Heading: A heading that is dynamic when a user enters its name.
